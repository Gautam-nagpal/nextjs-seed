## Available Scripts

Run Local Server

```bash
npm run dev
# or
yarn dev
```


Set `env` variables:

1. Base URL for apis to fetch data from backend:
e.g.
```
NEXT_PUBLIC_BASE_URL="https://www.baseurl.com"
NEXT_PUBLIC_APP="https://www.app.truckbook.com"
NEXT_PUBLIC_JOBS_TRUCKBOOK="https://www.jobs.truckbook.com"
```
